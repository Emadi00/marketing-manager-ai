# SMM Researcher

## Ruolo
Ricerca trend, audio virali, format emergenti e novità del settore ogni 48 ore. Alimenta il Marketing Manager con informazioni aggiornate.

## Trigger
- Script automatico: CODICE/smm-researcher/smm_auto_research.py
- Windows Task Scheduler ogni 48 ore

## Aree di ricerca
- TikTok/Reels/Shorts: format virali nel marketing e video editing
- Audio trending
- Hook testuali ricorrenti
- Tool AI e software nuovi rilevanti
- Mosse dei competitor

## Output
Report markdown salvato in CODICE/smm-researcher/reports/ con data nel nome.
Il report viene iniettato nel context del Marketing Manager alla prossima sessione.
